# Модернизация стилистики главной страницы

## Что было сделано

Адаптирована современная стилистика из HTML файла под структуру Dior Host.

### Созданные файлы:

1. **`frontend/components/home/HomePageModern.tsx`** - новая версия главной страницы с современной стилистикой:
   - Sticky навигация с backdrop blur
   - Hero секция с glow эффектами
   - Status bar с карточками
   - Современные карточки тарифов с hover эффектами
   - IPHM секция
   - Why Us секция
   - Testimonials
   - Comparison table
   - FAQ с details/summary

2. **`frontend/app/page-modern.tsx`** - альтернативная страница для тестирования новой версии

### Обновленные файлы:

1. **`frontend/tailwind.config.ts`** - добавлены новые цвета и анимации:
   - `surface`, `surfaceHighlight`, `primary`, `accent`
   - `hero-glow` градиент
   - `card-gradient` градиент
   - `animate-float` анимация

2. **`frontend/app/globals.css`** - добавлены кастомные стили:
   - Кастомный scrollbar
   - Стили для details/summary элементов

3. **`frontend/components/Header.tsx`** - обновлена навигация:
   - Добавлен `sticky top-0`
   - Добавлен `backdrop-blur-md`

## Как использовать

### Вариант 1: Заменить текущую версию (рекомендуется)

В файле `frontend/app/page.tsx` замените:

```tsx
import { HomePage } from "../components/home/HomePage";

export default function Page() {
  return <HomePage locale="en" content={homeContent.en} />;
}
```

на:

```tsx
import { HomePageModern } from "../components/home/HomePageModern";
import { homeContent } from "../content/home";

export default function Page() {
  return <HomePageModern locale="en" content={homeContent.en} />;
}
```

### Вариант 2: Тестировать новую версию на отдельной странице

Новая версия доступна по адресу: `/page-modern`

Можно переключиться, изменив маршрут или создав условный рендеринг.

## Особенности новой стилистики

- **Современный дизайн** с градиентами и glow эффектами
- **Плавные анимации** при скролле и hover
- **Улучшенная типографика** с правильным spacing
- **Адаптивность** для всех устройств
- **Accessibility** - правильная семантика и ARIA атрибуты
- **Производительность** - оптимизированные анимации через Framer Motion

## Что изменилось визуально

1. Hero секция - более просторная с glow эффектами
2. Статус карточки - более компактные с лучшей визуальной иерархией
3. Тарифы VDS - карточки с градиентными границами и hover эффектами
4. FAQ - использует нативные HTML details/summary элементы
5. Навигация - sticky с backdrop blur для лучшего UX

Все изменения совместимы с существующим контентом и структурой данных.
