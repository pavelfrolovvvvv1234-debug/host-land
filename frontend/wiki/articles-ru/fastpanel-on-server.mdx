# Установка FastPanel на сервер

FastPanel — современная панель управления для хостинга сайтов на VDS/VPS серверах.

## Что такое FastPanel?

FastPanel — это бесплатная панель управления веб-хостингом с удобным интерфейсом.

### Возможности:
- Управление сайтами
- Настройка доменов
- Управление базами данных
- SSL-сертификаты
- Мониторинг ресурсов

## Установка

### Требования:
- Чистая ОС Linux (Ubuntu 20.04+, Debian 10+, CentOS 7+)
- Минимум 1 GB RAM
- Root-доступ

### Команда установки:

```bash
bash <(curl -s https://fastpanel.direct/install.sh)
```

После установки панель доступна по адресу: `https://your-server-ip:8888`

## Первоначальная настройка

1. Откройте панель в браузере
2. Создайте администратора
3. Настройте DNS
4. Добавьте первый сайт

## FAQ

### Бесплатна ли FastPanel?
Да, FastPanel полностью бесплатная.

### Какие ОС поддерживаются?
Ubuntu, Debian, CentOS, AlmaLinux.

### Можно ли на Windows?
Нет, только Linux.
