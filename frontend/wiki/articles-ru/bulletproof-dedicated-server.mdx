# Абузоустойчивые выделенные серверы

Абузоустойчивый выделенный сервер (Bulletproof Dedicated Server) — это физический сервер в офшорном дата-центре, который не отключается автоматически при жалобах и DMCA-уведомлениях.

## Отличия от VDS

### Выделенный vs Виртуальный

**Выделенный сервер:**
- Физическое железо целиком в вашем распоряжении
- Нет соседей — все ресурсы только для вас
- Максимальная производительность
- Возможность кастомных настроек оборудования

**Виртуальный сервер (VDS):**
- Виртуальная машина на физическом сервере
- Ресурсы разделяются с другими VDS
- Дешевле выделенного
- Быстрое создание (минуты vs часы)

## Абузоустойчивость

### Как обрабатываются жалобы?

1. **Фиксация** — жалоба регистрируется в системе
2. **Анализ** — оператор проверяет обоснованность
3. **Уведомление** — клиент получает тикет
4. **Время на исправление** — без остановки сервера

### Что блокируется?

**Нулевая толерантность только к:**
- Child abuse (эксплуатация детей)
- Терроризм
- Malware распространение

Всё остальное обсуждается с клиентом.
