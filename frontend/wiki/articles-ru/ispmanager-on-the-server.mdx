# Установка ISPmanager на сервер

ISPmanager — профессиональная панель управления хостингом от компании ISPsystem для управления виртуальными и выделенными серверами.

## Что такое ISPmanager?

ISPmanager — коммерческая панель управления с расширенным функционалом:

### Возможности:
- Управление хостингом
- Почтовые серверы
- DNS-управление
- Базы данных
- FTP и файловый менеджер
- Мониторинг и статистика

## Версии ISPmanager

### ISPmanager Lite
Базовая версия для управления одним сервером.

### ISPmanager Business  
Расширенная версия с биллингом и автоматизацией.

## Установка

### Требования:
- VDS/Dedicated сервер
- ОС Linux (CentOS, Debian, Ubuntu)
- Минимум 2 GB RAM
- Root-доступ
- Лицензия ISPmanager

### Команда установки:

```bash
wget https://cdn.ispmanager.com/install && bash install
```

## Стоимость лицензий

- ISPmanager Lite: ~$10/мес
- ISPmanager Business: ~$30/мес

*Точные цены уточняйте на сайте ispmanager.com

## Первоначальная настройка

1. Установите панель
2. Откройте `https://your-server-ip:1500`
3. Войдите под root
4. Настройте параметры
5. Добавьте домены и сайты

## FAQ

### Бесплатна ли ISPmanager?
Нет, требуется платная лицензия.

### Какие ОС поддерживаются?
CentOS, Debian, Ubuntu, AlmaLinux.

### Можно ли на Windows?
Нет, только Linux.

### Есть ли пробный период?
Да, обычно 14 дней trial.
