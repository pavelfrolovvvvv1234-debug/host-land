# VPN: Инфраструктурное руководство

Virtual Private Network (VPN) — технология для создания защищённого подключения через интернет.

## Протоколы

- **WireGuard** — современный и быстрый
- **OpenVPN** — надёжный и проверенный
- **IPSec** — корпоративный стандарт

## Применение

- Корпоративный доступ
- Обход блокировок
- Защита приватности
- Безопасность на публичных сетях
