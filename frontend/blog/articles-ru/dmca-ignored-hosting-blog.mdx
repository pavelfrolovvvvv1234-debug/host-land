# DMCA-игнорируемый хостинг

DMCA-игнорируемый хостинг — сервис, который не отключает серверы автоматически при получении DMCA-уведомлений.

## Как работает?

1. DMCA-уведомление поступает провайдеру
2. Провайдер пересылает его клиенту
3. Клиент может ответить или игнорировать
4. Сервер продолжает работать

## Кто использует?

- Файлообменники
- Медиа-платформы
- Торрент-трекеры
- Контент-агрегаторы
